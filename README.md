## Описание пакета gs_interfaces

## Описание
Данный пакет предоставляет набор типов сервисов и сообщений, необходимых для работы Geoscan Pioneer Max

## Состав пакета:
1. Типы сообщений:
    * SimpleBatteryState.msg
    * SatellitesGPS.msg
    * PointGPS.msg
    * OptVelocity.msg
    * Orientation.msg
    * Parameter.msg

1. Типы сервисов:
    * Event.srv
    * Info.srv
    * Led.srv
    * Live.srv
    * Log.srv
    * Position.srv
    * PositionGPS.srv
    * Time.srv
    * Yaw.srv
    * NavigationSystem.srv
    * ParametersList.srv
    * SetNavigationSystem.srv
    * SetParametersList.srv
